# Archive

Centralized archive of past study-note repositories.  
⚠️ **Status:** Not maintained. Content is preserved only for reference.

## Contents
- **IPSUR/** — R scripts from G. J. Kerns’ *Introduction to Probability and Statistics using R (IPSUR)*.
- **MVA/** — small notes/scripts on Multivariate Analysis.
- **Resampling_New_Statistics/** — examples inspired by Julian L. Simon’s *Resampling: The New Statistics* (1997).
- **Miscellanea/** — isolated scripts for various statistical applications.

## Notes
- No active development.
- Original locations and dates are documented in each folder’s `ARCHIVE.md`.
